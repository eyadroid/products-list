# Frontend

Using Vue 3 and Typescript with Vite.


## Project Setup
Run install script.
```sh
npm install
```
then copy and set .env variables

```sh
cp .env.example .env
```

## Usage
### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
